N2T - Identifier Resolution Service

N2T is an identifier scheme resolver that given a provided identifier, matches it to an identifier scheme definition. Depending on the form of the request, a successful match will either redirect to the registered target or present information about the matched definition.

Identifiers take the form:

scheme:prefix/value

Note: The legacy N2T service is deprecated. The following resources are transitioned:

  • naan_registry.txt
  • shoulder_registry.txt
  • naan_table.txt

Information about ARK NAAN records is available at https://cdluc3.github.io/naan_reg_priv/

The N2T service API description is available at /api.

Identifier schemes registed with N2T are listed on the schemes page.
